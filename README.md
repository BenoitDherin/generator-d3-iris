# generator-d3-iris

## Getting Started

```
$ npm install -g yo
```

### Yeoman Generators

To install generator-d3-iris from npm, run:

```
$ npm install -g generator-d3-iris
```

Finally, initiate the generator:

```
$ mkdir <project-dir>
$ cd <project-dir>
$ yo d3-iris
$ gulp
```

### Create a new test case

```
$ yo d3-iris:new <name>
```
